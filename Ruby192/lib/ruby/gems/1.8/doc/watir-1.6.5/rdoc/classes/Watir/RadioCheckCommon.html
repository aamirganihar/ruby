Class Watir::RadioCheckCommon
In: lib/watir/input_elements.rb
Parent: InputElement

This class is the class for radio buttons and check boxes. It contains methods common to both. Normally a user would not need to create this object as it is returned by the Watir::Container#checkbox or Watir::Container#radio methods

most of the methods available to this element are inherited from the Element class
