new(datafile)

This allows you to put data in a csv file or an xls file and access it in your scripts for a data driven approach.

You can supply a file with extension csv or xls and it will just work

e.g. require ‘datahandler’ d = DataHandler.new("data.csv") loginId = d.data[0].LoginId —> line 1 of the csv file and column LoginId pin = d.data[0].Pin —> line 1 of the cvs file and column Pin

b = Datahandler.new("data.xls") loginId = b.data[0].LoginId —> line 1 of workbook 1 of the xls file with column LoginId

Remember that line 1 of the data is referenced by using 0 - and this excludes the header line.
