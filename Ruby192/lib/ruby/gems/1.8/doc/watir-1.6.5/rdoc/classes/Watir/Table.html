Class Watir::Table
In: lib/watir/table.rb
Parent: Element

This class is used for dealing with tables. Normally a user would not need to create this object as it is returned by the Watir::Container#table method

many of the methods available to this object are inherited from the Element class

to_a()

This method returns the table as a 2 dimensional array. Don‘t expect too much if there are nested tables, colspan etc. Raises an UnknownObjectException if the table doesn‘t exist. www.w3.org/TR/html4/struct/tables.html
