Selenium Ruby is a project that wraps selenium-rc into an object-oriented way, and wraps it into a RubyGem.

This rDoc is intended for you the check out the available methods on the objects. For a full document of what Selenium can do for you, please check out the home page: selenium.rubyforge.org

For document on the Selenium project and its RC, see Selenium homepage: www.openqa.org/selenium

Please post your question at mailing-list.
